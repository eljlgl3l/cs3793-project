## Setup

### Windows

```console
git clone <repo>
cd <repo>
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python train.py
```

### Unix
```shell
git clone <repo>
cd <repo>
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 train.py
```
